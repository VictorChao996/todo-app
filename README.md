## Description

### Intro
A simple todo app.
This is a warm up practice for another personal project.
I'm trying something new here first.

### tech stacks
- Frontened: Vue 3
- Backend: node.js, Express.js

## installation

### prerequistie
- git
- node

### install
```
git clone <<this-repo-link>>
```

### local dev
- frontend
```
cd frontend
npm install
npm run dev
```
- backend
```
cd backend
npm install
npm run dev
```

> for more executions see the package.json file
